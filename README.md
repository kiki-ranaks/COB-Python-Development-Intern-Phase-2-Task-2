# Chat Application with nicegui

This is a simple chat application built using Python and the `nicegui` library. It allows users to send and receive text messages in a chat-like interface.

## Features

- Real-time chat messages with avatars.
- User-friendly interface.
- Messages are displayed in chronological order.
- Press Enter to send a message.

## Installation

1. Make sure you have Python installed on your system.

2. Install the required dependencies using pip:

```bash
pip install nicegui
```
Run the application by executing the Python script:
```bash
python your_script.py
```
